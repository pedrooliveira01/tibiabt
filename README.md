# tibiabt
*Note: Este projeto não é um cavebot e sim somente um uma interação via mensagem com o cliente
utilizando as hotkey cadastrada no jogo

## Requisitos
* Delphi Berlin +

## Funções
* Selecionar o processo do tibia que sera vinculado
* Incluir acoes dos hotkey baseando na mana ou no HP ou no tempo
* Possui verificadores de manashield, haste, poison, paralize
